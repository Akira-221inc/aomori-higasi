## 概要

本作品は、**Unity**を用いて制作した**宇宙船内を舞台とするミニゲーム集**である。
宇宙船内で発生するさまざまなシステムエラーに対し、プレイヤーが制限時間内に対応する複数のミニゲームを実装した。

本制作ではテーマを**「再現性」**とし、同一条件下で常に同一の結果が得られるよう、ゲーム挙動の安定性と一貫性を重視して設計を行った。

---

## アピールポイント

* **AIをコード生成以外に使用していない点**
  アセットはフリー素材およびBlenderによる自作を中心に構成し、映像・音声素材も含めて制作過程そのものを重視した。

* **全員がUnity操作を経験したチーム開発**
  明確な役割分担のもと、ミニゲーム集という形式を採用することで、Unityの多様な機能を全員が試行錯誤しながら学習した。
  効率のみを重視すれば一部メンバーに実装を集中させる選択もあったが、本チームでは講義の目的であるスキル習得を重視し、全員が企画から実装までを経験する方針を取った。

* **教育的価値の高い開発体制**
  Unity経験者が1名のみという状況下でも、無理のない開発体制を構築し、チーム開発を通じた学習効果の高い制作を実現した。
